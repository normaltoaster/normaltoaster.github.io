MathJax Demonstration (No Style)

This page showcases how to render mathematical expressions using MathJax, without any custom CSS.

Inline Mathematics

Here's an example of an inline equation: The famous formula for the area of a circle is $A = \pi r^2$. You can also use the `\(...\)` delimiters, like so: The Pythagorean theorem states that \(a^2 + b^2 = c^2\). These are good for short expressions embedded within sentences.

Displayed Mathematics

For larger, more complex equations, it's often better to display them on their own line. Here's the quadratic formula using `$$...$$` delimiters:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

And here's another example, using the `\[...\]` delimiters for an integral:

\[ \int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2} \]

More Complex Examples (LaTeX Syntax)

Matrices

A 2x2 matrix can be written as:

$$ \mathbf{M} = \begin{pmatrix} a & b \\ c & d \end{pmatrix} $$

Systems of Equations

We can also represent systems of equations:

$$ \begin{cases} 2x + 3y = 7 \\ x - y = 1 \end{cases} $$

Limits and Sums

Consider the definition of a derivative using a limit:

$$ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} $$

And a simple sum:

$$ \sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6} $$

Fractions and Roots with Scaling Parentheses

Notice how the parentheses automatically scale to fit the fraction:

$$ E = mc^2 \left( 1 + \frac{p^2}{2m^2c^2} \right) $$

A cube root example:

$$ \sqrt[3]{x^3 + y^3} \neq x + y $$

How to View This File

  1. Save the code above into a file named `mathjax_no_style.html`.
  2. Open the `mathjax_no_style.html` file with your web browser.
  3. You'll see the mathematical expressions rendered by MathJax, using your browser's default styles.
